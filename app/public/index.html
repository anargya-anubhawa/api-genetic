Welcome to Genetic API!

Genetic API based from Pautres.

Pautres is a Web Server-based API that can be integrated into AutoResponder applications.

Select an API to see more information.

How to Use the API

  1. Add a new AutoResponder rule.
  2. In your AutoResponder rule, set the received messages that should be processed by your web server. It is usually recommended to use the All button in the upper right corner to process all messages with your web server.
  3. Select the Web Server option in your AutoResponder rule and enter the API URL.
  4. The Basic Auth and Headers options are optional. Please refer to the comments in the code above if you'd like to use them. It is recommended to add some kind of authorization if your API allows access to sensitive information.
  5. That's it! You are now ready to test your web server integration.

Genetic WhatsApp Bot

Genetic is an autoresponder-based WhatsApp bot with features like a Node.js bot. This bot was created by Anargya with contributions from Maxstream for answer rule inspiration and ItsReimau which supports a number of APIs used in GeneticBot.
